A modern Angular user dashboard with lazy-loaded user form, real-time role-based pie chart using Chart.js, and reactive state management via RxJS. Built with Angular Material & Bootstrap and deployed using Netlify.

## Features

- Material table to display users
- Lazy-loaded Add User dialog
- Reactive form validation
- Real-time pie chart using Chart.js
- RxJS BehaviorSubject for state management

## Tech Stack

- Angular
- Angular Material
- Bootstrap
- Chart.js
- RxJS

## Deployment

Deployed on Netlify with GitHub CI/CD.  
Every push to the `main` branch triggers an automatic deployment.
